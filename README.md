# Lazybones templates

## Templates

#### WordPress gradle

A lazybone template to startup a WordPress project

Install with:

	lazybones create wp-gradle 1.1 <projectdir>

## BinTray repository

Add the following configuration to: `~/.lazybones/config.groovy`

	bintrayRepositories = [
    	"stefanogualdi/lazybones-templates"
	]
